📊 Sales Performance Dashboard – Power BI
📌 Project Overview
This Sales Performance Dashboard was built in Microsoft Power BI to analyze sales performance across customers, products, months, and business segments. The dashboard provides interactive insights that help track key business metrics and support data-driven decision-making.
